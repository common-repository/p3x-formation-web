Affichez les dernières formations du Web sur votre site Internet (Ecoles, universités, centre de formation, etc...).

== Description ==

p3x est un site qui vous permettra de trouver tous les outils nécessaires pour la création de votre site Web. Sur celui-ci, vous trouverez toutes les formations du Web et de l'informatique.

Grâce au module Wordpress p3x, vous pourrez rapidement afficher sur votre site toutes les dernières formations informatique proposées par la plateforme p3x.
Ceci vous permettra de garnir votre contenu et de séduire vos visiteurs. En proposant plus de contenu en lien avec votre thématique, vous gagnerez surement plus de visiteurs.

== Installation ==

1. Téléchargez le plugin depuis votre administration Wordpress ou transférez le dans un nouveau dossier dans le répertoire `/wp-content/plugins/`
2. Activez le plugin depuis votre administration
3. Ajoutez vos formations en ajoutant le tag `[p3x_formation]` dans le contenu de vos pages
